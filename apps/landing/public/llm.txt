# NextClaw

Homepage: https://nextclaw.io/
Documentation: https://docs.nextclaw.io/
Repository: https://github.com/Peiiii/nextclaw
NPM Package: https://www.npmjs.com/package/nextclaw

Summary:
NextClaw is an OpenClaw-compatible personal AI gateway with a web UI.
It supports multi-provider model routing, multi-channel delivery, and local-first operation.
